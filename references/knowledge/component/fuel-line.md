# Fuel Line（燃油管路）零部件知识

## 典型 BOM 结构

```
Fuel Line Assembly
├── Feed Line（供油管）
│   └── Monolayer/Multilayer Nylon Pipe (SAE J2260)
├── Return Line（回油管）
│   └── Nylon Pipe
├── QC Connectors（快速接头）
│   ├── Female 90° SAE 5/16"
│   └── Female 90° SAE 3/8"
│   └── 金属材质，二次锁止（PF.90298 / SAE J2044）
├── Metal Pipe（金属管段，如有）
│   └── 电焊钢管 9.02145/02，内镀镍
├── Damper（阻尼器）
│   └── 工作压力范围 3-6 bar
├── Anti-abrasion Sleeve（防磨套管）
│   └── PF.90271
├── Protection Caps（保护帽）
│   └── 所有管端必须加，运输防污染
└── Clips（卡扣）
    └── 装配到车身/管路插入
```

## 测试矩阵概要

| 测试类别 | 关键测试项 | 标准 |
|----------|----------|------|
| 环境 | 化学浸泡、CaCl2、ZnCl2、腐蚀 | PF.90197 §5 |
| 物理/机械 | 拉伸、接头拔脱、卡扣力、尺寸 | PF.90197 §6 |
| 功能 | 清洁度、泄漏、压降、爆破 | PF.90197 §7 |
| 安全 | 碰撞(FMVSS 301)、渗透(MiniSHED) | PF.90197 §8 |
| 耐久 | 生命周期、振动、脉冲压力 | PF.90197 §9 |

样件数量参考 PF.90068 Annex A，需与 OEM 规范工程师协商确定。

## 处理注意事项

1. **多条件分行**：爆破压力有 3 个条件（RT/高温/化学浸泡后），脉冲耐久有 2 个条件（30万/60万），必须每个条件独立一行
2. **金属管镀镍**：如 BOM 含金属管段，内镀镍是常见要求，需确认电镀能力
3. **阻尼器外购**：阻尼器通常由指定供应商提供，需确认采购渠道
4. **CaCl2 区域差异**：仅 EMEA 市场适用，需确认项目目标市场
5. **清洁度分阶段**：DV 阶段仅柴油管路需合规，PV 阶段全部管路
6. **碰撞测试分工**：FMVSS 301 整车测试由 OEM 负责，供应商配合
7. **100% 泄漏测试**：在线泄漏测试是产线必备设备，需 15µm 虚拟泄漏检测能力
8. **保护帽必须**：所有管端必须加保护帽，消除需 OEM 和工厂 UTE 双方同意
9. **虚拟数据处理**：RFQ缺失CDS/开发计划时可虚拟填充但需标注

## DEF 管路（PF.90068）注意事项

部分 Stellantis Fuel Line RFQ 同时含 DEF（尿素）管路：
- 受 PF.90068（37页，Heated Diesel Exhaust Fluid Supply Lines）独立约束
- 材料需耐尿素溶液腐蚀；加热功能需电气资质
- DV 样件数和测试矩阵与 PF.90197 不同，需分开制作 SVP
- 如属首次处理，需评估 DEF 管路制造经验和学习成本

## OEM 特有要求

### Stellantis
- 性能标准主文件：PF.90197（Fuel Bundle, 28页, Harmonized）
- DEF 管路有独立规范 PF.90068
- QC 接头独立规范 PF.90298
- SVP 格式：07740 Annex 2（FCA 格式）
- 质量目标：CCP 0.00@3月 / 0.035@12月, ICP=0
- 可靠性：P99C90（功能）+ R95/C90（耐久）
- 3D 要求：Native UG + TcAE，不接受其他格式
- 供应商集成等级通常为 Level 2
