## 文档组合模式

Stellantis 发 Fuel Filter 类零件的 RFQ 时，文件包典型组成：

```
RFQ 文件包（实际收到）
├── PF.90150 — 性能标准（44页，PDF）
│   └── 覆盖环境/物理/机械/电气/功能/可靠性/耐久/CC 全部测试要求
└── CTS — Component Technical Specification（15页，PDF）
    └── 特定应用参数 + 8个嵌入附件（PDF中不可提取）

期望但缺失的文件
├── SSTS / SOR — 顶层技术规范
├── TDR — 交付物清单及模板（5个 Excel 模板嵌入其中）
├── Deliverables 时间表
├── 3D CAD 数据
├── Product Card（传感器参数）
└── CDS（Component Data Sheet）
```

**关键发现**：该组合下 TDR 和 SSTS 经常缺失，需主动向客户索取。CTS 的 PDF 版本无法提取嵌入附件（DV/PV 模板、FDM、保护帽规格等），需索要原始 Office 版本。

## 典型例外项

该组合下反复出现的例外事项：

| 例外项 | 说明 |
|--------|------|
| CAD/TcAE 能力 | Stellantis 要求 UG + TcAE，非通用 CAD 格式 |
| EMC 测试实验室 | 需 Stellantis 认可实验室，选择有限 |
| SENT 协议能力 | NTC 传感器使用 SENT 而非模拟信号 |
| PTC 加热器设计/测试 | 加热器子系统跨电气/热力学，验证项多 |
| 指定连接器供应商 | APTIV 连接器为指定件 |
| 传感器集成能力 | 需同时集成 NTC + WiF 两类传感器 |
| 乳化水分离测试 | 需专用设备（Carl Fisher 分析仪） |
| 100% 在线测试设备 | 外密封 + 传感器功能 + 接头尺寸，产线投资大 |

## 成本参考范围

- 材料成本中，加热器和传感器组件占比显著（>40%）
- 测试成本高：28 项 DV/PV + 13 项 CC，耐久测试样件 45 pcs/组
- 标准购买成本：约 12 项 ISO/SAE/EN 标准需购买
- 在线测试设备投资：至少 3 个专用工位（密封/传感器/尺寸）
- 开发周期：加热器寿命测试（5000 cycles）是关键路径

## 历史踩坑记录

1. **CTS PDF 嵌入文件不可提取**：CTS 以 PDF 格式提供时，8 个嵌入附件（DV/PV 模板、FDM、保护帽规格等）均无法提取。必须向客户索要原始 Office 版本的 CTS 或单独提供附件
2. **CTS vs PF.90150 参数冲突**：水分离效率在 CTS 中为 ≥93%，在 PF.90150 中为 ≥95%。规则：CTS 应用参数优先于通用标准，但需明确标注双源
3. **Product Card 依赖**：NTC 和 WiF 传感器的具体参数（特性曲线、精度要求等）在 PF.90150 中仅为示例值，实际参数由 Product Card 定义。Product Card 常缺失
4. **Smart Filter 跨学科复杂度**：集成 5 个功能子系统（过滤+水分离+PTC加热+NTC温度+WiF水位），需机械/电气/传感器多学科协同，FMEA 复杂度高
5. **无 TDR 时交付物推断**：使用 Stellantis 标准 15 项 TDR 交付物 + CTS 引用 + PF.90150 要求进行推断，但可能遗漏客户特有格式要求
