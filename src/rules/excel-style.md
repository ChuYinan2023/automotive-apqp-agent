# Excel 生成规则

所有 `.xlsx` 文件使用 `openpyxl` 生成，统一遵循以下样式规范。

## 字体

- 全局字体：Arial
- 标题行：加粗

## 颜色规范

| 用途 | 背景色 | 字色 | 说明 |
|------|--------|------|------|
| 标题行 | `#4472C4`（蓝） | `#FFFFFF`（白） | 所有表头 |
| 人工填写项 | `#FFF2CC`（黄） | 默认 | 需要人手动填写的单元格 |
| 公式/自动计算 | `#C6EFCE`（绿） | 默认 | 公式自动计算的单元格 |
| 阻塞项/例外项 | `#FFC7CE`（红） | 默认 | 例外事项、阻塞项 |
| 待确认项 | `#BDD7EE`（蓝） | 默认 | 需要确认的单元格 |
| 辅助/只读 | `#E7E6E6`（灰） | 默认 | 参考信息 |
| CC 特性行 | `#FFE6E6`（粉红） | 默认 | Critical Characteristic |
| SC 特性行 | `#FFF2CC`（浅黄） | 默认 | Significant Characteristic |
| 冲突行 | `#FFD966`（黄） | 默认 | 数据冲突标记 |

## 缺口优先级颜色

| 优先级 | 颜色 |
|--------|------|
| 最高/高 | `#FF7F7F`（红） |
| 中 | `#FFD966`（黄） |
| 低 | `#C6EFCE`（绿） |

## 格式规范

- 边框：全部 thin
- 冻结首行
- 列宽：根据内容自适应（最小 10，最大 50）
- 数字格式：百分比用 `0.0%`，货币用 `#,##0.00`

## 使用前提

生成时需先激活项目虚拟环境（确保 openpyxl 可用）。
