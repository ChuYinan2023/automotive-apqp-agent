# 模板优先级规则（铁律）

生成每项交付物前，**必须**按以下顺序判断使用哪个模板。一旦确定路径，该项目内保持一致。

## 优先级顺序

### 1. 客户提供了模板 → 用客户模板

- **来源**：`out/_嵌入文件/` 中提取的客户标准模板
- **判断标准**：文件名含 `模板`/`template`/`TDR_` 且为客户提供；或在 `out/分析/客户模板清单.md` 中已标注
- **操作**：用 openpyxl 读取客户模板，定位填写区域，注入数据
- 🛑 **不改客户的列头、sheet 名、格式风格**

### 2. 客户给了交付物清单但无模板 → 自主生成

- **来源**：客户 TDR 中列出了交付物名称但未提供模板文件
- **操作**：用 openpyxl 按 `rules/excel-style.md` 的规则直接生成

### 3. 两者都没有 → 问用户

- **操作**：问用户"需要生成哪些交付物？"，确认后用 openpyxl 直接生成

## 检查方式

交付物生成 Agent 在生成每个交付物前，必须：

1. 查看 `out/分析/客户模板清单.md` 中该交付物是否有对应客户模板
2. 有 → 走路径 1
3. 无 → 检查客户是否在 TDR 中列出了该交付物
4. 有列出 → 走路径 2
5. 未列出 → 走路径 3

## 最终一致性检查关联

`checklist/final-consistency.md` 第 6 项"模板合规"会验证此规则是否被正确执行。
