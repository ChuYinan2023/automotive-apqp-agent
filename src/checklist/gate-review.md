# Gate Review 检查清单

## 执行时机

交付物全部生成、一致性检查全部 PASS、知识回写完成后，执行 Gate Review。

## 检查项

### 文档完整性

- [ ] README.md（结构化分析报告）已生成且用户已确认
- [ ] 缺失项跟踪表已生成，"最高"优先级项均有处理方案
- [ ] 全部客户要求的交付物已生成
- [ ] 最终一致性检查 8/8 PASS

### 技术评估

- [ ] 特殊特性（CC/SC）已完整提取并经用户确认
- [ ] 质量目标已提取并经用户确认
- [ ] 例外事项已识别，每项有用户决策（接受/替代方案/委外）

### 风险评估

- [ ] 技术风险已识别并汇总
- [ ] 商务风险已识别并汇总
- [ ] 质量风险已识别并汇总
- [ ] 所有高风险项有应对方案

### 商务准备

- [ ] 成本分解（CBS）已完成
- [ ] 报价文件格式符合客户要求
- [ ] 开发计划时间节点可行（用户确认）

### 知识回写

- [ ] OEM 知识已回写
- [ ] 零部件知识已回写
- [ ] 组合经验已回写
