# Fuel Supply Module（燃油供给模块总成）零部件知识

## 典型 BOM 结构

```
Fuel Supply Module ASSY (Fördereinheit)
├── Electric Fuel Pump（电动燃油泵）
│   ├── EC 泵（三相换向，FSI/MPI/TDI 高效方案）
│   └── DC 泵（单相 PWM，MPI 基础方案）
├── Fuel Pump Electronics / KPE（燃油泵电子控制单元）
│   ├── EC 驱动：三相换向，PWM 9-25 kHz
│   └── DC 驱动：单相 PWM 电压控制
├── Fuel Level Sensor / Tankgeber（油位传感器）
│   ├── Lever-type（杠杆式）— 电阻式
│   └── TKN（厚膜传感器）— 替代方案
├── Suction Jet Pump / Saugstrahlpumpe（射流泵）
│   └── 鞍型油箱双腔供油
├── Fuel Fine Filter（燃油精滤器）
│   └── ISO 19438 验证
├── Extraction Strainer / Absaugkäfig（抽吸滤网）
├── Leakage Protection Valves / Leckage-Schutzventile（泄漏保护阀）
│   └── 安全件 — 碰撞/断开条件下零泄漏
├── Pressure Limitation Valve / Druckbegrenzungsventil（压力限制阀）
│   └── 安全件 — 开启 620-640 kPa
├── Pressure Holding Valve / Druckhalteventil（保压阀）
│   └── 仅汽油系统 — 停机后维持残压
├── Parking Heater Stop Valve / Standheizungsventil（驻车加热器截止阀）
├── Module Housing / Bracket（模块壳体/支架）
├── Electrical Connector（电气连接器）
├── Seals / O-Rings（密封件）
└── Fuel Line Fittings（管路接头）
```

## 处理注意事项

1. **多条件分行**：泵效率有 DC/EC/柴油三个条件，压力有开启/关闭两个条件，油位传感器有满/空/总三个条件 — 每个条件必须独立一行
2. **品牌差异**：VW 集团版本中油位传感器电阻值 VW 与 Audi/Porsche 不同，需注意品牌标注
3. **系统变体复杂**：4种架构（FSI闭环/FSI开环/MPI/TDI）意味着多套参数和测试矩阵
4. **安全件标注**：泄漏保护阀和压力限制阀是安全件(CC)，需特别关注
5. **KPE双模**：EC+DC双泵型支持增加软件和测试复杂度
6. **耐久周期长**：8000小时温度-时间循环约需12个月，是项目关键路径
7. **FAM-B燃油**：耐久和老化试验需使用 FAM-B 标准测试燃油
